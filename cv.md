## Code example:

### Remove the minimum

```
function removeSmallest(numbers) {
  let arr = [];
  let minValue = numbers.indexOf(Math.min.apply(null, numbers));
  numbers.forEach((elem, index) => {
    if (index !== minValue) {
      arr.push(elem);
    }
  });
  return arr;
}

```
